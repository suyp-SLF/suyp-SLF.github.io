# 可以参考预览
https://suyp-slf.github.io/3DModelHtml/
## 显示3D模型，简单版
```
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的 3D 模型展示</title>
    <style>
        body { margin: 0; background-color: #f0f0f0; display: flex; justify-content: center; align-items: center; height: 100vh; }
        /* 设置查看器的大小 */
        model-viewer {
            width: 80%;
            height: 80%;
            background-color: #ffffff;
            border-radius: 15px;
            box-shadow: 0 10px 30px rgba(0,0,0,0.1);
        }
    </style>
</head>
<body>

    <script type="module" src="https://ajax.googleapis.com/ajax/libs/model-viewer/3.4.0/model-viewer.min.js"></script>

    <model-viewer 
        src="alien.glb" 
        alt="一个3D模型" 
        auto-rotate 
        camera-controls 
        shadow-intensity="1" 
        environment-image="neutral" 
        exposure="1"
        ar>
    </model-viewer>

</body>
</html>
```
